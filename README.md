**電影專案練習**

1. 標籤切換 (最新活動、免費活動、熱門活動)
2. 無限捲動讀取
3. 使用 immutable.js
4. 必須實作 process reducer 處理 API 發送狀態 (不能用 loading flags)
5. redux-api-middleware
6. API timeout for saga (optional)
